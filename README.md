# TrabajoPractico_nro_3
Trabajo practico nro 3 para python Nicolas Gregorio, TuPrimeraPagina+Gregorio

.Objetivo del proyecto

Desarrollar una web en Django utilizando el patrón MVT

Utilizar herencia de plantillas HTML

Crear al menos 3 modelos

Implementar formularios para insertar datos

Implementar un formulario de búsqueda en la base de datos

Subir el proyecto completo a GitHub

. Modelos utilizados

La aplicación gimnasio contiene los siguientes modelos:

Socio: nombre, email, fecha de inscripción

Entrenador: nombre y especialidad

Clase: nombre, entrenador, horario y cupo

Estos modelos están definidos en:

gimnasio/models.py

. Funcionalidades principales:

Alta de socios, entrenadores y clases mediante formularios

Visualización de clases disponibles

Búsqueda de socios por nombre

Administración completa desde el panel /admin de Django

Uso de herencia de plantillas (base.html)

Uso de archivos estáticos (CSS e imagen de fondo)

-Tecnologías utilizadas:

Python 3

Django 6.0

HTML

CSS

SQLite (base de datos por defecto de Django)
